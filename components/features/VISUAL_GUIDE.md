## Color Coding

### Light Mode
- **Blue** - Data sources, information
- **Purple** - Methodology, calculations
- **Indigo** - About, mission
- **Green** - Success, data quality
- **Amber** - Warnings, disclaimers
- **Red** - Errors, critical info

### Dark Mode
- Uses `dark:` variants throughout
- Neon accents: cyan (#00f5ff), purple (#b537f2), pink (#ff2d95)
- Background layers: primary, secondary, tertiary
- Text hierarchy: primary, secondary, tertiary

## Component Sizes

### DataSources Compact
- Height: ~40px (single line)
- Width: Flexible (inline-flex)
- Best at: Top of sections

### DataSources Full
- Height: ~800px
- Width: 100% (max-w-7xl)
- Best at: Standalone section

### MethodologyModal
- Height: 90vh (scrollable)
- Width: max-w-4xl
- Best at: Modal overlay

### AboutSection
- Height: ~900px
- Width: 100% (max-w-7xl)
- Best at: Bottom of page

### Footer
- Height: ~400px
- Width: 100% (full-width)
- Best at: Bottom (auto)
